== Description ==

Displays a "Miniplan" on Wordpress sites and let privileged people edit it. Use the [miniplan id='x'] tag.

== Installation ==

standard

 - Install this plugin through wordpress plugin manager.

 - Activate the plugin in admin-plugin section

 - add [miniplan id='x'] to sites

github

 - clone the repository from github.com/sedrubal/WP-Miniplan to `/wp-content/plugins/WP-Miniplan`

 - Activate the plugin in admin-plugin section

 - add [miniplan id='x'] to sites
